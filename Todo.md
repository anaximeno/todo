# To Do CLI

## Use cases:
* [x] `todo add <name> [-d "description"] [-t "task 1" "..." "task n"]`
* [ ] `todo show <name:taskid?>`
* [ ] `todo drop <name:taskid?>`
* [ ] `todo done <name:taskid?>`
* [ ] `todo update <name> -n <new-name> -d ["new description"]`
* [ ] `todo update task <taskid> "new task"`
